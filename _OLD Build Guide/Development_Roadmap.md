## 📝 Notes & Considerations

### Design Philosophy
- **Satirical Horror:** Balance dark themes with absurd humor
- **Retro Aesthetic:** Commit fully to 16-bit pixel art limitations
- **Player Agency:** Choices in Phase 2 meaningfully affect gameplay
- **Subversive Mechanics:** "Observe" button satirizes pet care games

### Technical Considerations
- Canvas-based rendering for pixel-perfect control
- State machine for phase management
- LocalStorage for save/persistence
- Responsive design for mobile compatibility

### Future Enhancements
- Achievement system
- Speedrun mode
- Alternate endings
- Easter eggs referencing deep Alien lore
