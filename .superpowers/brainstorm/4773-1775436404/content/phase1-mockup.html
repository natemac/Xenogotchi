Key design decisions this raises:

Q1

Should each egg's block column have a different block count / durability?

E.g. Obsidian = most blocks (dense, hard to crack), Crimson = medium+packed, Ivory = fewest but spread out. This gives each egg a distinct "feel" without changing the core Breakout mechanic.

Q2

Keep the "warmth particles fall to egg" visual idea?

The spec says broken blocks release warmth particles that drift down to the egg below. This is a nice visual that ties block-breaking to egg progress. Worth keeping as a juice/feedback element.

Q3

Keep the "direct hit on egg" as the final win condition?

The spec requires: clear all your egg's blocks AND then hit the egg directly to hatch it. This two-step win gives a satisfying "finishing blow" moment. Or: auto-hatch when all blocks clear?
