Q1

Block count = egg difficulty?

Obsidian = most blocks (13), Crimson = medium (10), Ivory = fewest (7).
More blocks = longer to hatch, but more warmth particles hit the egg.
Keep this asymmetry or make all equal?

Q2

Warmth particles — keep or cut?

Spec says broken blocks release particles that drift down to the egg.
Nice visual feedback. Worth implementing or too complex for v1?

Q3

"Direct hit to hatch" — keep or auto-hatch?

Spec requires: clear all blocks → then land a direct hit on the egg.
Satisfying "killing blow" moment. OR: auto-hatch when last block clears?

Q4

Fail state: blocks regenerate or lives system?

Current spec: miss ball 3× → some blocks partially regenerate (setback, not game over).
Alternative: classic 3 lives, lose one on each miss, game resets Phase 1 on 0 lives.
